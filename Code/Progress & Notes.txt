Notes:


For linking files in html lets use these formats;
                                to link files in same directory or in sub directory;
                                                                ./file_name
                                                                ./sub-directory_name/file_name

                                to link files in different directory;
                                                                /Code/directory_name/file_name
                                                                /Code/directory_name/sub-directory_name/file_name

Let's use these formats and don't use complete file paths. This will try to avoid conflict between mac and windows users when linking files.
